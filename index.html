London Phonegap

Let"s hack your phone with some HTML5 and a cup of tea !

This group is for developers interested in mobile app development in Phonegap using Javascript frameworks such as AngularJS and Ember.js. All skill and experience levels are welcome. The main aim is to discuss techniques and share experiences over a friendly beverage. Sometimes this will involve watching a presentation.

Past events

16/06 - June 2014 meetup (The night meetup)

  • Verifying a PhoneGap build with Smoke Tests & Appium by Ezo Saleh

    Ezo showed us how to test PhoneGap apps with smoke tests and Appium. Appium is an open source test automation framework for use with native and hybrid mobile apps. It drives iOS and Android apps using the WebDriver JSON wire protocol. No more excuse for testing your apps!

  • Why JavaScript devs should learn Swift? by James Nocentini

    James presented Swift, the new Apple language for iOS & Mac OS and how building Phonegap plugins will be easier with it.
